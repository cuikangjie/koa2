# koa2 demo

## 使用

```cmd
 1.克隆项目
  git clone  https://github.com/cuikangjie/koa2.git

 2. 删除目录下 .git

 3. use for you

```

## 开发

- 安装依赖 yarn install

- 项目启动 yarn run dev

- 浏览器访问 http://localhost:9090/

## 生产环境(项目使用pm2启动，可以修改pm2目录下config)

- 打包 yarn build

- 上传build目录下的所有文件

- 安装依赖 yarn install

- 项目启动 yarn run start
